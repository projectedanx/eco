## Setup & Installation

**Prerequisites:**  Node.js (v18+ recommended) and Python 3.

1. **Clone the repository** and navigate to the root directory.
2. **Install frontend dependencies:**
   `npm install`
3. **Environment Configuration:**
   Create a `.env.local` file based on `.env.example` (if present) or manually set your Gemini API Key:
   `echo "GEMINI_API_KEY=your_api_key_here" > .env.local`

## Usage

* **Run the Development Server:**
  Use the package manager to start the vite dev server.
  `npm run dev`

  *The app will be accessible at http://localhost:3000. It provides a visual dashboard of the ECO Protocol features, loaded agents, and the active Lexicon.*

* **Build for Production:**
  `npm run build`

* **Run Epistemic Simulations:**
  To validate the CoC (Chain of Thought) Enactment and RCC-8 Paraconsistent states:
  `python3 aew_simulation.py`
